# Little Library
Little Library is an app to collect little libraries.
This is a group project with Jeff Holtzkener.

## Features
Map | Login | Sign Up | Logout | Search

## Setup

Install dependencies with `npm install`.

## Running Webpack Development Server

```sh
npm start
```
